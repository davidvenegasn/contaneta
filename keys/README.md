## `keys/` (NUNCA se versiona)

Este directorio es para llaves/certificados (por ejemplo **FIEL** `.key`/`.cer`).

- **No se deben commitear** llaves ni certificados.
- En desarrollo puedes colocar aquí tus archivos para pruebas locales.
- En producción deben existir fuera del repo (volumen/secret manager) y con permisos restrictivos.
